# Aether: The Minimalist Writing Space

Focused environment for distraction-free writing.

## Tech Stack
- Pure HTML5
- Vanilla CSS3
- No frameworks, no external libraries.

## Design Principles
- Typography first.
- High contrast and meaningful whitespace.
- Minimalist aesthetics.
- Rapid prototyping iteration (25 versions).

## Build Instructions
- Each version is a self-contained HTML file in the `versions/` directory.
- `index.html` at the root is the gallery.

## Style Preferences
- Modern, clean, and premium typography.
- No generic colors; use curated HSL palettes.
- Smooth transitions and hover effects.
